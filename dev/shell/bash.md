# Learning Bash #

## Basics ##
1) For linux/mac give permission to the file
2) hash(#) + bang(!) = shebang(#!)

**Give Permision**
```bash
    sudo chmod +x start.sh 
        or
    sudo chmod 777 start.sh
```

**Run the file**
```bash
    ./[name-of-the-file]
    Ex: ./myscript.sh
```

**Which Bash**
```bash
    which bash
```
**Output** :: /usr/bin/bash
* Varies with users and operating system
