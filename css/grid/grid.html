Display Grid

Grid layout with rows and columns, making it easier to design web pages without having to use floats and positioning
Grid needs to have a display: grid to its parent The vertical lines of grid items are called columns The horizontal lines of grid items are called rows. The spaces between each column/row are called gaps. The lines between columns are called column lines. The lines between rows are called row lines. Grid-template-columns property defines the number of columns in your grid layout, and it can define the width of each column. The grid-column property defines on which column(s) to place an item.
